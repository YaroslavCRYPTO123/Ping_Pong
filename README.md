🕹️ Игра Ping-Pong (Пинг-понг)
Динамичный прототип классической аркады, созданный на Python с помощью библиотеки Pygame. Проект демонстрирует базовые принципы игровой физики, обработки столкновений и управления объектами в реальном времени.
🚀 Особенности игры
Автоматика: Мяч перемещается по сцене самостоятельно, отскакивая от границ игрового поля.
Управление: Поддержка двух игроков для управления левой и правой ракетками с клавиатуры.
Физика: Реализована логика отскока мяча от ракеток и боковых стенок.
Условие проигрыша: Игра фиксирует поражение игрока, который не успел отбить мяч своей ракеткой.
Гибкость: Код подготовлен для легкого расширения функционала и добавления новых идей.
