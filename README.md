🏓 Игра Ping-Pong (Прототип)
Классическая аркада, написанная на Python с использованием библиотеки Pygame. Прототип включает в себя базовую механику столкновений и управления для двух игроков.
🚀 Особенности игры
Управление: Движение ракеток с помощью клавиатуры (W/S для левого игрока и стрелки для правого).
Автоматизация: Полностью автономное движение мяча по сцене с расчетом траектории отскока.
Физика: Реализована логика столкновений мяча с границами экрана и ракетками игроков.
Условие проигрыша: Автоматическое определение победителя, если один из игроков пропускает мяч.
