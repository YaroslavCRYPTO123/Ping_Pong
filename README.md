Прототип игры «Пинг-понг»
Упрощенная реализация классической игры "Пинг-понг", созданная на языке Python с использованием библиотеки Pygame.
📋 Описание проекта
Данный проект является техническим прототипом, в котором реализованы базовые механики игрового процесса. Основной упор сделан на логику движения объектов и обработку пользовательского ввода.
Реализованный функционал:
Автоматическое движение мяча: мяч перемещается по сцене и отскакивает от границ.
Управление ракетками: два игрока могут управлять своими ракетками с клавиатуры.
Условие завершения: игра определяет проигрыш, если один из игроков пропускает мяч.
